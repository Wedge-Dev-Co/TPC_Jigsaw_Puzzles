# Puzzle Images

Drop image files here to add them as puzzles.

**Supported formats:** .jpg, .jpeg, .png, .webp

**Naming:** The filename (without extension) becomes the puzzle ID and title.
- `mountain-lake.jpg` -> Puzzle titled "mountain lake"
- `sunset_beach.png` -> Puzzle titled "sunset beach"

**Recommended size:** 800x600 to 1920x1080 pixels. Landscape orientation works best.
